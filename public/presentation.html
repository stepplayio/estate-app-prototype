시스템

  • 통계
  • 관리자

1단계: 잠재매물 (Pre-Farming)

구현 완료

  • 지도 뷰 / 리스트 뷰 전환
  • 지역별 필터 (강남/서초/송파 등)
  • 단계별 필터 (Pre-Farming/Farming/등록승인)
  • 출처별 필터 (국토부/네이버/수동)
  • Signal 점수 시각화
  • 수동 입력 / 엑셀 업로드

자동화 포인트

  • 국토부 API 자동 수집 (월 1회)
  • 네이버 크롤링 (주 2~3회)
  • Signal 점수 자동 계산
  • 주거용 자동 필터링

2단계: 활동 관리 (Farming)

구현 완료

  • 활동 등록 (간편 폼, 모바일 최적화)
  • 활동 관리 (목록, 필터)
  • 일정 캘린더 (권한별)
  • 활동 유형별 필터 (방문/통화/내방)
  • 사진/음성 녹음 표시
  • 활동 통계 (건수, 전환율)

자동화 / 구현 예정

  • 일정 알림 자동 발송 (D-1, 당일)
  • 사진 업로드
  • 음성 녹음 (STT + GPT 요약)
  • 조사표 작성 (엑셀 기반)
  • 현재 위치 자동 기록

3단계: 물건 접수 (Sales Offering)

구현 완료

  • 접수 목록 (카드 뷰)
  • 매도인 정보 관리
  • 서류 체크리스트 (진행률 표시)
  • 상태별 필터 (서류 대기/접수 완료)
  • 4단계 전환 버튼

자동화 / 구현 예정

  • 2단계 데이터 자동 연결
  • 서류 업로드 (R2)
  • 필수 서류 자동 체크
  • OCR 자동 분석 (Phase 2)

4단계: 매물 등록 (Register) ⭐ 중요

구현 완료

  • 결재 워크플로우 시각화 (에이전트→팀장→경영진)
  • 품질 점수 표시 (70점 기준)
  • 승인/보완요청/반려 버튼 (권한별)
  • 결재 대기/승인 완료 필터
  • 결재 관리 페이지 (팀장/관리자 전용)

자동화

  • 3단계 데이터 자동 연결
  • 품질 점수 자동 계산
  • 결재 알림 자동 발송
  • 5단계 자동 전환 (승인 완료 시)

5단계: 마케팅 (Marketing) ⭐ 중요

구현 완료

  • IM 편집 모달 (실제 포맷 반영)
  • 사용자 입력: 사진 2장, 입지 특성
  • 자동 생성: 건축물대장, 수익률, 예상 월매출
  • 체크리스트 진행률 표시
  • VIP 발송 현황 표시

자동화 / 구현 예정

  • 4단계 데이터 자동 연결
  • 수익률 자동 계산 (공식 적용)
  • PDF 자동 생성
  • 메일 자동 발송 (VIP)
  • 체크리스트 자동 생성

6단계: 매수자 관리 (Buyer Contact)

구현 완료

  • 잠재/확정/영업 중 상태 구분
  • 매수자 목록 (2단 레이아웃)
  • 매칭 결과 표시 (점수순)
  • 조건 확정하기 (잠재→확정 전환)
  • 영업 시작 버튼 (7단계 전환)
  • 등록 매물 관리 (매물별 매칭 현황)

자동화

  • 매칭 로직 자동 실행
  • 점수 자동 계산 및 정렬
  • 7단계 자동 연결

7~9단계: 영업 / 계약 / 사후관리

Phase 2 또는 추후 개발

현재 프로토타입은 1~6단계 중심으로 구현
7단계(영업), 8단계(계약), 9단계(사후관리)는 개발 계약 후 진행

7단계: 영업 관리

  • 영업 파이프라인 (칸반 뷰)
  • 영업 활동 기록
  • 피드백 관리
  • 일정 알림

8단계: 계약 관리

  • 계약서 작성
  • 전자서명 (Phase 2)
  • RTMS 신고 (Phase 2)

9단계: 사후 관리

  • 사후관리 일정
  • 만족도 조사
  • 고객 리포트 (Phase 2)

프로토타입 핵심 구현 사항

대시보드

  • 9단계 프로세스 현황 (건수)
  • 데이터 최신 업데이트 (국토부/네이버)
  • 알림 드롭다운 (헤더)

관리자 페이지

  • 배치 Job 관리 (실행 이력)
  • 스케줄/이벤트 Job 구분
  • 수동 실행 버튼
  • 성공률 통계

권한별 메뉴

  • 에이전트: 활동 중심
  • 팀장: 팀 전체 + 결재
  • 관리자: 전체 + 시스템
  • 사이드바 그룹화 (서브메뉴)

추가 기능

  • 통계 페이지 (구현 예정 카드)
  • 일정 캘린더 (플레이스홀더)
  • 등록 매물 관리 (매칭 현황)
